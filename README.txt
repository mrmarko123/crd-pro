# 🧬 CRD Pro — تحلیل حرفه‌ای طرح کاملاً تصادفی

**نسخه ۳.۰**  
پلتفرم تحت وب برای تحلیل آماری طرح‌های کاملاً تصادفی (CRD)  
مناسب برای محققان، دانشجویان و اساتید زیست‌شناسی، کشاورزی، پزشکی و علوم پایه

---

## 🚀 امکانات

- **ورود دستی داده‌ها** با پیش‌نمایش زنده و اعتبارسنجی لحظه‌ای  
- **آنالیز واریانس یک‌طرفه (One-Way ANOVA)**  
- **بررسی پیش‌فرض‌های آماری**: نرمالیتی (Shapiro-Wilk) و همگنی واریانس (Levene)  
- **مقایسه‌های چندگانه Tukey HSD** با ماتریس p-value و حروف معنی‌داری (a, b, c)  
- **نمودارهای تعاملی**: Box Plot، نمودار میله‌ای با خطای استاندارد، Q-Q Plot  
- **خروجی CSV و Excel** از داده‌ها و جداول تحلیل  
- **رابط کاربری شیشه‌ای (Glassmorphism)** با فونت فارسی ایران‌سنس و تم تیره  
- **پشتیبانی از حداقل یک گروه و یک مشاهده** (با نمایش آمار توصیفی و نمودار)
